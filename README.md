# EzheGheler - Advanced Socket Programming

Aplikasi Full-Stack Node.js untuk mendemonstrasikan implementasi Socket Programming tingkat lanjut menggunakan perpaduan protokol **TCP (Transmission Control Protocol)** dan **UDP (User Datagram Protocol)**.

## Fitur Utama
1. **Sistem Login Klien:** Dilengkapi dengan verifikasi SMTP email (Nodemailer).
2. **Transfer Berkas TCP:** Mentransmisikan data biner (file video/media) ke server secara handal melalui *pure socket* (Port 3001).
3. **Broadcasting Media UDP:** Menyiarkan paket datagram secara *real-time* ke klien yang mendengarkan di Port 4000.
4. **Dashboard Control Panel:** UI berdesain *Glassmorphism* modern dengan animasi metrik jaringan.

## Cara Menjalankan
1. Pastikan Anda telah menginstal [Node.js](https://nodejs.org/).
2. Jalankan perintah `npm install` untuk mengunduh semua modul.
3. Jalankan server dengan perintah `node server.js`.
4. Akses dashboard di browser melalui `http://localhost:3000`.
